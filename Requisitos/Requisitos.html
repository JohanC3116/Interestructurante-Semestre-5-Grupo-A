Reto EIE - TikTok: Entre Diversión y Dependencia

Descripción del Reto

El proyecto busca enfrentar la creciente adicción a la plataforma TikTok, especialmente entre los jóvenes. Su algoritmo adictivo y el diseño de la interfaz incentivan el uso compulsivo, lo cual afecta la salud mental, la productividad y el bienestar emocional de los usuarios. El reto consiste en desarrollar una solución tecnológica que permita a los usuarios tener un control consciente sobre su uso de TikTok y otras redes sociales, promoviendo así hábitos digitales más saludables.

Objetivo General

Desarrollar una aplicación que ayude a los usuarios a controlar el uso de TikTok mediante estadísticas de uso, recomendaciones personalizadas, alertas y recompensas, promoviendo así un uso más saludable y equilibrado de las redes sociales.

Objetivos Específicos

  • Definir los requerimientos del sistema con enfoque en la gestión del tiempo en redes sociales.
  • Diseñar una base de datos que almacene información sobre el uso de TikTok y otras redes.
  • Generar notificaciones personalizadas que promuevan el uso consciente.
  • Desarrollar recomendaciones y consejos basados en los patrones de uso recolectados.
  • Implementar un sistema de recompensas para incentivar el uso equilibrado de redes sociales.

Requisitos Funcionales

  • El sistema debe registrar el tiempo de uso de TikTok.
  • Debe enviar notificaciones cuando se excedan los límites establecidos.
  • Debe permitir configurar metas de uso personalizadas.
  • Debe generar reportes visuales con estadísticas de uso.
  • Debe ofrecer recomendaciones personalizadas de uso saludable.
  • Debe permitir el registro de usuarios con perfil básico.

Requisitos No Funcionales

  • La aplicación debe ser accesible desde dispositivos móviles Android y iOS.
  • Debe garantizar la privacidad de los datos del usuario.
  • Debe contar con una interfaz intuitiva y amigable.
  • El sistema debe tener alta disponibilidad y bajo tiempo de respuesta.

Requisitos de Información

  • Información del usuario (nombre, correo electrónico, edad).
  • Registro de tiempo de uso por día y por aplicación.
  • Preferencias del usuario sobre límites de uso y notificaciones.
  • Historial de reportes y recomendaciones ofrecidas.
